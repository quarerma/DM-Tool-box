# DM-Musicbox

**DM-Musicbox** is a lightweight audio control tool for TTRPG Dungeon Masters, built to manage soundtracks in real time without disrupting gameplay.

It allows smooth transitions between tracks, flexible looping, and quick access to organized music, helping you shift seamlessly between scenes like exploration, combat, and roleplay.

## Features

- Real-time track switching  
- Smooth fades and transitions  
- Configurable audio loops  
- Scene-based organization  
- Custom soundboard with hotkeys  

Designed for speed and simplicity, DM-Toolbox keeps your focus on storytelling while maintaining full control over your game’s atmosphere.
